“使用hexo搭建博客”

使用hexo搭建博客

  1. 安装node.js（hexo基于node），网上可直接安装.完成后利用”node -v”，”npm -v”查看版本号,知晓是否安装完成。

  2. 执行“npm install -g cnpm –registry=https://registry.npm.taobao.org”命令，利用npm，淘宝地址，安装cnpm，因为国内的镜像慢，故利用cnpm

  3. 执行“cnpm”，查看是否安装完毕

  4. 执行“cnpm install -g hexo-cli”，安装hexo（可利用“hexo -v查看版本信息）

  5. 新建个空文件夹，在他的路径下打开cmd，运行“hexo init”命令。（ps：确保这个命令能够执行的前提是需要安装git，否则无法进行。网上可直接下载）如执行在某处卡顿不前，可以将文件删了，重新来过，本人试了第二次，很快成功。

  6. 使用“hexo server”运行hexo

  7. 在浏览器上运行localhost：4000，出现如下页面表示成功，如无法显示，则返回第一步重新开始

  8. 使用hexo new ”我的第一篇博客文章“，新建一篇博客。可在给出的目录下查看是否新建成功。
